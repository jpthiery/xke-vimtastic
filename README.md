# xke-vimtastic

## First backup your existing Vim config

```
mv ~/.vimrc ~/.vimrc.bak
mv ~/.vim ~/.vim.bak
```

## Copy this Vim configuration

```
cp .vimrc ~/.vimrc

```

## Install Vundle

```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
## Install Plugins

```
vim +PluginInstall +qall
```
